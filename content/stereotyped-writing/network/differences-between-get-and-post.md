# GET和POST有什么区别 <Badge text="字节" type="info" />

|              |                                GET                                 |             POST              |
| :----------: | :----------------------------------------------------------------: | :---------------------------: |
| 参数传递方式 |       通过URL查询字符串传递：https://example.com/api?page=0        | 通过请求体（RequestBody）传递 |
| 参数长度限制 | 有长度限制（HTTP协议中没规定长度限制，但各浏览器实现中有长度限制） |         没有长度限制          |
|     缓存     |                         GET请求可以被缓存                          |        POST不会被缓存         |
|     场景     |                              获取数据                              |         提交表单数据          |
